## New features

### VertexAI embedder

New `VertexAIEmbedder` for embedding with Google Vertex AI:

```typescript
import { VertexAIEmbedder } from '@toolpack-sdk/knowledge';

const embedder = new VertexAIEmbedder({
  project: 'your-gcp-project',
  location: 'us-central1',
  model: 'text-embedding-005',
  retries: 3,
});
```

- Configurable retry count with per-attempt error logging.
- Supports batch embedding.
- Works alongside existing `OpenAIEmbedder` and `OllamaEmbedder`.

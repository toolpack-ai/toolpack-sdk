# Toolpack SDK Samples

This directory contains sample projects demonstrating various integrations with the Toolpack SDK.

## 📁 Available Samples

### [toolpack-sdk-integrations](./toolpack-sdk-integrations/)

**A comprehensive collection of 15+ minimal Node.js examples** covering all major integration patterns.

#### What's Included:
- ✅ **Basic Providers** - OpenAI, Anthropic, Gemini, Ollama
- ✅ **Multi-Provider** - Dynamic switching, model discovery
- ✅ **Custom Providers** - Implement any LLM API
- ✅ **Tools Integration** - Built-in and custom tools
- ✅ **Modes & Workflows** - Autonomous agents
- ✅ **Streaming** - Real-time responses
- ✅ **Multimodal** - Image processing
- ✅ **Embeddings** - Semantic search
- ✅ **Event-Driven** - Status tracking
- ✅ **Advanced Config** - Production patterns

#### Quick Start:
```bash
cd toolpack-sdk-integrations
export TOOLPACK_OPENAI_KEY="sk-..."
npm run example:openai
```

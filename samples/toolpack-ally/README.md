# Toolpack Ally

A minimal Electron desktop companion app that keeps Toolpack resources close at hand.

## Getting started

1. Install dependencies (only required the first time):
   ```bash
   npm install
   ```
2. Start the desktop app:
   ```bash
   npm start
   ```

## Environment helpers

- Set `ELECTRON_DEVTOOLS=true` before running `npm start` to automatically open Chrome DevTools when the window loads.
